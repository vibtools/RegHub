# RegHub v0.2.0 Environment Variables

All v0.1.1 production variables remain valid. New integrations are optional and default to disabled
or unauthenticated public access.

## Existing required production values

```env
APP_NAME=RegHub
APP_ENV=production
APP_DEBUG=false
PUBLIC_BASE_URL=https://reghub.ygit.dev
ALLOWED_HOSTS=["reghub.ygit.dev","localhost","127.0.0.1"]
CORS_ORIGINS=["https://ygit.net","https://www.ygit.net"]
DATABASE_URL=postgresql+asyncpg://...
DATABASE_POOL_SIZE=5
DATABASE_MAX_OVERFLOW=5
SESSION_SECRET=...
SESSION_COOKIE_SECURE=true
SESSION_MAX_AGE_SECONDS=28800
OIDC_ISSUER_URL=https://auth.vib.tools/realms/YOUR_REALM
OIDC_CLIENT_ID=reghub
OIDC_CLIENT_SECRET=...
OIDC_SCOPES=openid profile email
OIDC_ADMIN_CLAIM=roles
OIDC_ADMIN_VALUES=["reghub-admin"]
OIDC_END_SESSION_URL=...
GITHUB_TOKEN=...
GITHUB_TIMEOUT_SECONDS=15
GITHUB_ALLOW_PRIVATE_REPOSITORIES=false
PUBLIC_API_CACHE_SECONDS=60
LOG_LEVEL=INFO
```

## GitLab

```env
GITLAB_TOKEN=
```

Public GitLab imports work without a token but are rate-limited. Use a company-managed token only
when private repository access or higher limits are required.

## Bitbucket

```env
BITBUCKET_USERNAME=
BITBUCKET_APP_PASSWORD=
```

Use a company-managed Bitbucket service identity/app password. Public imports work without them.

## Provider timeout

```env
PROVIDER_TIMEOUT_SECONDS=20
```

## Optional AI enrichment

```env
AI_METADATA_ENABLED=false
AI_BASE_URL=
AI_API_KEY=
AI_MODEL=gpt-4.1-mini
```

Deterministic metadata is always generated. AI failures never block imports and never replace the
registry security rules.

## Optional screenshot service

```env
SCREENSHOT_SERVICE_URL=
SCREENSHOT_SERVICE_TOKEN=
```

The service must accept `POST {"url":"https://preview"}` and return
`{"screenshot_url":"https://..."}`. It must run outside RegHub. RegHub never launches a browser or
executes template code.

## Local manifest/ZIP imports

```env
LOCAL_UPLOAD_ENABLED=false
LOCAL_UPLOAD_MAX_BYTES=25000000
LOCAL_UPLOAD_MAX_UNCOMPRESSED_BYTES=100000000
LOCAL_UPLOAD_MAX_ENTRIES=2000
```

Keep disabled unless administrators need local imports. Uploaded ZIPs are inspected in memory and
not executed. No persistent volume is required for v0.2.0.

## Coolify secret flags

OIDC_CLIENT_SECRET

GITLAB_TOKEN
BITBUCKET_APP_PASSWORD
