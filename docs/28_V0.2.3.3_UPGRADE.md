# RegHub v0.2.3.3 Upgrade

1. Back up the current source and record the healthy v0.2.3.2 Coolify deployment.
2. Extract the replace-ready archive and copy the `reghub` contents over the repository root.
3. Preserve `.git`, `.env`, PostgreSQL, Keycloak, and all Coolify variables.
4. Commit, push, and redeploy normally.

No Alembic migration is included. Existing runtime settings, encrypted credentials, API service
tokens, block rules, templates, and operation history remain unchanged.

After deployment, open `/admin/settings` and test one action in every tab. The selected tab must stay
open, the page must remain responsive, and no request should return HTTP 404.
