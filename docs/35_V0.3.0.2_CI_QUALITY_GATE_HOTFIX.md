# v0.3.0.2 CI Quality Gate Hotfix

The v0.3.0.1 GitHub Actions run failed in `ruff check .` before the remaining CI jobs could execute. The reported findings were static quality-gate failures rather than a demonstrated application runtime failure. Because tests, migration verification, and Docker smoke checks were blocked, v0.3.0.1 should not be promoted until CI is green.

This hotfix changes only import ordering, unused imports, and test lint directives. No database or runtime behavior changes.
