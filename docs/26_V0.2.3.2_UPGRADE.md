# RegHub v0.2.3.2 Upgrade

1. Back up PostgreSQL and record the current healthy v0.2.3.1 deployment.
2. Extract the replace-ready ZIP in a temporary directory.
3. Copy the contents of its `reghub` directory over the current repository root.
4. Preserve `.git`, local `.env`, Coolify variables, `SESSION_SECRET`, Keycloak configuration, and PostgreSQL.
5. Commit, push, and redeploy normally in Coolify.

No new migration is included. The existing entrypoint may run `alembic upgrade head`; it remains a no-op when the database is already current.

## First checks

- Open `/admin/settings#integrations-pane`, save an integration, and confirm the page does not reload or leave the Integrations panel.
- Open `/admin/settings#api-manage-pane` and check the **Original repository** endpoint.
- Open a completed operation and confirm logs start at the first line and remain compact.
- Verify `GET /api/v1/templates/{published-slug}/repository`.
