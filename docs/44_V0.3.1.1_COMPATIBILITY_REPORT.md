# RegHub v0.3.1.1 Compatibility Report

## Baseline

- Source baseline: v0.3.1.0
- Release type: Final Stabilization Hotfix
- API version: `/api/v1` unchanged
- Database reset: not required
- Manual migration: not required
- New environment variables: none
- New dependencies: none

## Preserved behavior

- Public API paths and typed response models
- OIDC and administrator authentication
- SQLAdmin pages and controls
- Draft, Publish and Disable lifecycle
- GitHub, GitLab, Bitbucket and local import behavior
- Source synchronization
- Assets and manifests
- Framework, language, package-manager, license, README, topics and repository analysis
- Dockerfile, CI, entrypoint and Coolify deployment

## Intentional data cleanup

Only obsolete analyzer-generated deployment fields and obsolete quality/evidence dimensions are
removed from historical analysis JSON. Administrator-curated manifest overrides are not modified.

## Upgrade result

Existing users can upgrade through the normal replace, push and Coolify deploy workflow. Existing
template IDs, slugs, publication states, repository metadata, assets and valid registry analysis are
preserved.
